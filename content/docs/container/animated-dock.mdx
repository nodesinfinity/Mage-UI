## Installation

<Steps>
<Step>Install dependencies</Step>

```bash
npm install framer-motion lucide-react
```

<Step>Run the following command</Step>

It will create a new file `animated-dock.tsx` inside the `components/mage-ui/container` directory.

```bash
mkdir -p components/mage-ui/container && touch components/mage-ui/container/animated-dock.tsx
```

<Step>Paste the code</Step>

Open the newly created file and paste the following code:

```jsx file=<rootDir>/mage-ui/container/animated-dock.tsx

```
