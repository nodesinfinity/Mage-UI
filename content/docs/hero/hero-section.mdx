## Installation

<Steps>

<Step>Update `tailwind.config.js`</Step>
```js
theme: {
    extend: {
      keyframes: {
          fadeIn: {
          from: { opacity: "0" },
          to: { opacity: "1" },
        },
      },
    },
    animation: {
      fadeIn: "fadeIn 0.5s ease-in",
    },
  },
```

<Step>Run the following command</Step>

It will create a new file called `hero-section.tsx` inside the `components/mage-ui/hero` directory.

```bash
touch components/mage-ui/hero/hero-section.tsx
```

<Step>Paste the code</Step>

Open the newly created file and paste the following code:

```jsx file=<rootDir>/mage-ui/hero/hero-section.tsx

```

</Steps>
