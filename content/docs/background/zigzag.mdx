## Installation

<Step>Update `tailwind.config.js`</Step>
```js
theme: {
    extend: {
     colors: {
        background: "hsl(var(--background))",
     }
    },
  },
```

<Steps>
<Step>Run the following command</Step>

It will create a new file called `zigzag.tsx` inside the `components/mage-ui/background` directory.

```bash
mkdir -p components/mage-ui/background && touch components/mage-ui/background/zigzag.tsx
```

<Step>Paste the code</Step>

Open the newly created file and paste the following code:

```jsx file=<rootDir>/mage-ui/background/zigzag.tsx

```

</Steps>
