## Installation

<Steps>

<Step>Update `tailwind.config.js`</Step>
```js
theme: {
    extend: {
     colors: {
        foreground: "hsl(var(--foreground))",
     }
    },
  },
```

<Step>Run the following command</Step>

It will create a new file called `mirror-text.tsx` inside the `components/mage-ui/text` directory.

```bash
mkdir -p components/mage-ui/text && touch components/mage-ui/text/mirror-text.tsx
```

<Step>Paste the code</Step>

Open the newly created file and paste the following code:

```jsx file=<rootDir>/mage-ui/text/mirror-text.tsx

```

</Steps>
