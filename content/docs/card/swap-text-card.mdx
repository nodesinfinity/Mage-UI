## Installation

<Steps>
<Step>Install dependencies</Step>

This uses [SwapText](/docs/text/swap-text) for the text. Install it by following the instructions [here](/docs/text/swap-text#installation).

<Step>Run the following command</Step>

It will create a new file called `swap-text-card.tsx` inside the `components/mage-ui/card` directory.

```bash
mkdir -p components/mage-ui/card && touch components/mage-ui/card/swap-text-card.tsx
```

<Step>Paste the code</Step>

Open the newly created file and paste the following code:

```jsx file=<rootDir>/mage-ui/card/swap-text-card.tsx

```

</Steps>
