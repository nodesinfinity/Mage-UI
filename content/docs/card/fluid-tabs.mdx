## Installation

<Steps>
<Step>Install dependencies</Step>

```bash
npm install framer-motion lucide-react
```

<Step>Update `tailwind.config.js`</Step>

Add the following to your tailwind.config.js file.

```json
module.exports = {
  theme: {
    extend: {
    }
  }
}
```

<Step>Run the following command</Step>

It will create a new file `fluid-tabs.tsx` inside the `components/mage-ui/card` directory.

```bash
mkdir -p components/mage-ui/card && touch components/mage-ui/card/fluid-tabs.tsx
```
