## Installation

<Steps>

<Step>Update `tailwind.config.js`</Step>
```js
theme: {
    extend: {
       keyframes: {
         "bg-position": {
         "0%": { backgroundPosition: "0% 50%" },
          "100%": { backgroundPosition: "100% 50%" },
         }
     }
    },
  },
```

<Step>Run the following command</Step>

It will create a new file called `avatar-list.tsx` inside the `components/mage-ui/list` directory.

```bash
mkdir -p components/mage-ui/list && touch components/mage-ui/list/avatar-list.tsx
```

<Step>Paste the code</Step>

Open the newly created file and paste the following code:

```jsx file=<rootDir>/mage-ui/list/avatar-list.tsx

```

</Steps>
