## Installation

<Step>Install dependencies</Step>

```bash
npm install lucide-react
```

<Steps>
<Step>Run the following command</Step>

It will create a new file called `menu-animation.tsx` inside the `components/mage-ui/list` directory.

```bash
mkdir -p components/mage-ui/list && touch components/mage-ui/list/menu-animation.tsx
```

<Step>Paste the code</Step>

Open the newly created file and paste the following code:

```jsx file=<rootDir>/mage-ui/list/menu-animation.tsx

```
