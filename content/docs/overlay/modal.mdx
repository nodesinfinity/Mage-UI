## Installation

<Steps>
<Step>Install dependencies</Step>

```bash
npm install framer-motion lucide-react
```

<Step>Run the following command</Step>

It will create a new file called `modal.tsx` inside the `components/mage-ui/overlay` directory.

```bash
mkdir -p components/mage-ui/overlay && touch components/mage-ui/overlay/modal.tsx
```

<Step>Paste the code</Step>

Open the newly created file and paste the following code:

```jsx file=<rootDir>/mage-ui/overlay/modal.tsx

```
