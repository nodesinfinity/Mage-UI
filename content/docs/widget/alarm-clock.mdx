## Installation

<Steps>

<Step>Update `tailwind.config.js`</Step>
```js
theme: {
    extend: {
     colors: {
        foreground: "hsl(var(--foreground))",
     }
    },
  },
```

<Step>Install dependencies</Step>

```bash
npm install lucide-react
```

<Step>Run the following command</Step>

It will create a new file called `alarm-clock.tsx` inside the `components/mage-ui/widget` directory.

```bash
mkdir -p components/mage-ui/widget && touch components/mage-ui/widget/alarm-clock.tsx
```

<Step>Paste the code</Step>

Open the newly created file and paste the following code:

```jsx file=<rootDir>/mage-ui/widget/alarm-clock.tsx

```

</Steps>
